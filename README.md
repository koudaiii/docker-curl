Docker Alpine Curl
--

Alpine image(TIMEZONE Asia/Tokyo) with ops tool installed.

- curl
- bash
- postgresql-client
- tcpdump
- redis-cli
- jq
- dig
- memcached-tool
- perl

Available from docker hub as [koudaiii/curl](https://hub.docker.com/r/koudaiii/curl/)

## Usage

```shell-session
$ docker run -it koudaiii/curl bash
```
